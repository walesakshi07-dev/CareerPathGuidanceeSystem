# CareerPathGuidanceeSystem
A web-based Career Path Guidance System developed using Python Flask, HTML, CSS and MySQL to provide skill-based career recommendations and learning guidance.
# 🚀 Career Path Guidance System

## 📌 Project Overview

Career Path Guidance System is a web-based application developed using Python Flask. This system helps students and users to choose suitable career paths based on their skills and interests.

The application provides career recommendations, skill management, course suggestions, and learning guidance. It helps users understand required skills and explore different career opportunities.

The main purpose of this project is to provide digital career guidance and support users in making better career decisions.

---

## 🎯 Project Aim

The aim of this project is to develop an intelligent career guidance platform that helps students identify suitable career options and provides learning resources according to their skills.

---

## 🎯 Objectives

- To provide career guidance digitally.
- To recommend careers based on user skills.
- To manage user skills and career information.
- To provide course and learning suggestions.
- To reduce confusion while choosing a career path.
- To help students improve required technical skills.

## ⚙️ Backend Implementation

- Developed using Python Flask framework.
- Flask routes are used to handle different modules.
- MySQL database is connected with Flask application.
- User authentication is managed using sessions.
- CRUD operations are implemented for managing users, skills and careers.

---

## 🗄 Database Tables

### Users Table

Stores user information.

Fields:
- User_ID
- Name
- Email
- Password


### Skills Table

Stores user skills.

Fields:
- Skill_ID
- User_ID
- Skill_Name


### Careers Table

Stores career details.

Fields:
- Career_ID
- Skill
- Career_Name
- Description


### Admin Table

Stores admin login information.

Fields:
- Admin_ID
- Username
- Password

---

## 🔄 Working Process

1. User registers into the system.
2. User logs in using email and password.
3. User adds skills.
4. System analyzes user skills.
5. Career recommendations are displayed.
6. User explores related courses and learning topics.
7. Admin manages career and user information.
